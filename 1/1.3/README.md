quiz
1. data is anything that holds information (information that can be moved, processed, or stored on a computer)
2. value is data (anything that can be represented as data)
3. an object is a type of variable (region of memory that stores a value)
4. variable is an identifier that stores data (object that has a name); identifier is the name
5. identifier is the name of variable
6. data type is used for variables (determines what kind of value (char, string, int, float) an object can store)
7. 4 byte number like 1, 2, 3 ... (whole number without decimal)
