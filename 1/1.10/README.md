1. a statement has a `;` and an expression just evaluates something (statements are for when the computer wants to perform an action/ instruction, expressions are for when the program wants to calculate a value)
2. 
a.
```c++
int x;
```
expression statement (statement of variable declaration)

b.
```c++
int x = 5;
```
expression statement (statement contains an expression)

c.
```c++
x = 5;
```
expression statement

d.
```c++
foo();
```
expression statement (function calls are an expression)

e.
```c++
std::cout << x;
```
statement contains an expression (expression statement: `std::cout` is the left operand, `<<` is the operator, `x` is the right operand)
